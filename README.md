# Expense Tracker

Simple expense tracker app. Login with any username/password, add expenses, see totals, delete if needed. Data saves in browser per user.

## What it does
- Add expenses (name, ₹ amount, category, date)
- Stats: total spent, count, biggest expense
- Delete with confirm
- Local storage - your data stays

## How to use
1. Open index.html
2. Login (anything works)
3. Add some expenses
4. Check stats

That's it. Made with HTML/JS/CSS.

Enjoy tracking! 💰
